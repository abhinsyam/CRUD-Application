# CRUD Application - stores details like Name, Email, Phone Number, Address, Status / Role (Lead, Permanent..etc), Entry creation time to 
the SQL database. 


# http://localhost:8080/ --> SHOWS WELCOME PAGE, With option to view "clients" which displays data stored in database. 

# URL http://localhost:8080/clients --> Displays details of every user in list format with the options to "Edit", "Delete" entry. 
User can edit and update the details in database as well as remove entry with ease. 

Thymeleaf, html, css used for the table view and display of database information.
